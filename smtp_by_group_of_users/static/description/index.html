SMTP By User

This module will allow to configure outgoing mail server by user.

Outgoing mail server configuration by user

- Added field named User
- You can create the outgoing mail server by specified by selected user.
- Here by selecting user, every email will be sent with reference to selected user, rather than default odoo mail system.
