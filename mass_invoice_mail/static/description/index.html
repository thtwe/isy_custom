Mass Invoice: Send By Mail

An application which allows to send mass invoice mail to the customers.

Features:

  • Send mass invoice mail
  • Confirmation message before sending invoices.
  • Flexibility to select template before sending invoices
  • User can send e-mail of separate as well as multiple invoices
  • Customer will receive email directly inside his/her mailbox with attachment

Mass e-mail action

Simplest way to send mass e-mail to customers using this action.

Different Template

User can select template before sending invoices.

Confirmation

Confirmation message before sending invoices.

Success/Error Message

Get brief message after sending invoice to the customers.

Customer E-mail

Customer get e-mail directly to his/her mail box with attachment
