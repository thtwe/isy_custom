User Guide

Setting Up the Module:

After installing the app, navigate to 'Settings' to reach the 'Users' menu under 'Users & Companies' and set the users you would like to have the ability to view login trials, passwords, and unblock users. (notice that only users with Settings access rights are able to do, view or edit any of what's mentioned before).

Navigate to 'Settings' all the way through, 'General Settings' to get to the 'Login Security' group and set the amount of time needed before lifting blocks on any user. (Such practice protects your system from malicious hackers).

Login Trials:

Try logging in as a user (ie: Marina) multiple times whether successful or not then logged in as one of the privileged users (ie: Mitchell), navigate to 'Settings' to reach the 'Login Trials' menu under 'Users & Companies'.

You can only view the password of failed attempts (attempts using a wrong password) but not the successful attempts or those of blocked users using correct passwords or restricted locations (IP Adresses) and you can also track & check who viewed login trial records as they are registered in the added chatter.

Login Locations:

You can register locations (IP Addresses) by navigating to 'Settings' to reach the 'Login Locations' menu under 'Users & Companies' and creating them.

You can add an extra level of security to your system by having users log in only from certain IP Addresses through setting locations on their user profile.

Blocked Users:

You can easily view blocked users by navigating to 'Settings' to reach the 'Users' menu under 'Users & Companies' and checking their status on the list view whether they were internal or portal users.

For a blocked user to be able to log into the system regardless of set block period & any other restrictions applied, navigate to the user's profile in 'Settings' and click the 'Unlock Login' button which only appears in cases of blocked users.
